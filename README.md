# Destinos de Águia

Arquivos de identidade visual da marca Destinos de Águia.

## Cores
- Azul Marinho: #1E3A5F
- Dourado/Bege: #E8B98A

## Tipografia
- Fonte: Afrah (Bold, Regular, Light)

## Arquivos
- Logos em fundo azul marinho, preto e bege
- Símbolo isolado da águia
- Apresentação completa da marca (PDF)
- Fontes: afrah-bold.otf, afrah-regular.otf, afrah-light.otf

## Sobre
Marca de conteúdo de viagens criada por Fernanda (@fersavieira).
O símbolo é inspirado na Águia Pescadora, representando liberdade,
visão ampla e múltiplos destinos.
